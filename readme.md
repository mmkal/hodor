# Hodor

[![Build Status](https://travis-ci.org/mmkal/hodor.svg?branch=master)](https://travis-ci.org/mmkal/hodor) [![Coverage Status](https://coveralls.io/repos/github/mmkal/hodor/badge.svg)](https://coveralls.io/github/mmkal/hodor)

> Hodor.

This is a programming language implementation, written in TypeScript. There's no formal spec for the language, and (as far as I know) this is the only implementation.

Here's what you should know, so you can get started using Hodor:

1. All keywords are some variant of `Hodor`.
1. All strings are composed out of variants of `Hodor`.

## Usage

```
npm install hodor
```

### Language

#### Via CLI

```
npx hodor path/to/file.hodor
```

The above command will run a file written in `hodor`.

#### Repl

```
npx hodor
```

This will start a repl, useful for getting to know hodor. Type any string in the prompt, and it will be translated into `hodor`-text for you.

You can also input a valid `hodor` expression, and it will be evaluated.

### Via API

```js
import { hodor } from "hodor";

hodor(
  `'Hodor Hodor, Hodor, Hodor. Hodor Hodor, Hodor. Hodor Hodor. Hodor, Hodor. Hodor!'("HODOR HODOR HODOR HODOR. Hodor. Hodor Hodor, Hodor Hodor. Hodor Hodor, Hodor Hodor. Hodor, Hodor, Hodor! Ho-dor!, Ho-dor? HODOR HODOR, HODOR! Hodor, Hodor, Hodor! Hodor Hodor, Hodor. Hodor Hodor, Hodor Hodor. Hodor, Hodor Hodor. Ho-dor!!");`
);
```

This program will print `"Hello, World!"`.

## Strings

Strings in Hodor are first translated to Morse Code, then "dots" become `Hodor` and "dashes" become `Hodor,`. At the end of a letter (sequence of dots and dashes), the `Hodor`s are punctuated with either a full-stop (if the sequence ends with a dot), a question mark (if the sequence has an even number of dots/dashed, and ends with a dash), or an exclamation mark (if the sequence has an odd number of dots/dashes, and ends with a dash). Capital letters are the same, but the `Hodor`s are capitalised. Words are separated by either `Ho-dor?` or `Ho-dor!`, depending on whether the last character was made up out of letters, or other characters.

For example, the string `'Hodor'` in Hodor is:

```
HODOR HODOR HODOR HODOR. Hodor, Hodor, Hodor! Hodor, Hodor Hodor. Hodor, Hodor, Hodor! Hodor Hodor, Hodor.
```

The string `'Why have you done this?'` in Hodor is:

```
HODOR HODOR, HODOR! Hodor Hodor Hodor Hodor. Hodor, Hodor Hodor, Hodor? Ho-dor! Ho-dor? Hodor Hodor Hodor Hodor. Hodor Hodor? Hodor Hodor Hodor Hodor? Hodor. Ho-dor! Ho-dor? Hodor, Hodor Hodor, Hodor? Hodor, Hodor, Hodor! Hodor Hodor Hodor! Ho-dor! Ho-dor? Hodor, Hodor Hodor. Hodor, Hodor, Hodor! Hodor, Hodor. Hodor. Ho-dor! Ho-dor? Hodor! Hodor Hodor Hodor Hodor. Hodor Hodor. Hodor Hodor Hodor. Ho-dor!?
```

## Numbers

Numbers in Hodor are encoded using Morse code, similar to strings. Numbers start with `Hodor...` (three dots) followed by the Morse code encoding of the number spelled out as words.

Numbers are spelled out digit by digit:
- Each digit is written as its word name (zero, one, two, three, four, five, six, seven, eight, nine)
- Digits are separated by spaces
- Decimal points are written as "point"
- Negative numbers start with "minus"

For example:
- `1` becomes `Hodor...` followed by Morse code for "one"
- `123` becomes `Hodor...` followed by Morse code for "one two three"
- `1.23` becomes `Hodor...` followed by Morse code for "one point two three"
- `-123` becomes `Hodor...` followed by Morse code for "minus one two three"

The number `123` in Hodor is:

```
Hodor...Hodor, Hodor, Hodor! Hodor, Hodor. Hodor. Ho-dor! Ho-dor? Hodor! Hodor Hodor, Hodor! Hodor, Hodor, Hodor! Ho-dor! Ho-dor? Hodor! Hodor Hodor Hodor Hodor. Hodor Hodor, Hodor. Hodor. Hodor.
```

## Writing Hodor Programs

Writing Hodor is somewhat similar to writing JavaScript or Python, if you imagine JavaScript or Python only using the word "Hodor". Identifiers are written similarly to strings, but use single quotes instead of double quotes. For example, a function called `foo` could be called like so:

```
'Hodor Hodor Hodor, Hodor. Hodor, Hodor, Hodor! Hodor, Hodor, Hodor!'()
```

### Keywords

| keyword | value   |
| ------- | ------- |
| If      | Hodor?  |
| Then    | Hodor!  |
| Else    | Hodor!! |
| Lambda  | Hodor   |
| True    | HODOR   |
| False   | hodor   |

Mathematical operators like `+`, `-`, `*`, `/` and `%` are all supported similarly to in languages like JavaScript. See samples below for how to use them.
